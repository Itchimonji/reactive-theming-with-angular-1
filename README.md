# Code reuse in Angular

This projects contains a little example for changing yours stylesheets dynamically with Angulars reactivity.

## Prerequisites

OS: any

Framework: Angular

Knowledge: Beginner

## Installation

Run following commands in the root directory

```
npm install
```

```
ng serve
```
